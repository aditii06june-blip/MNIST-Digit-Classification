# MNIST Digit Classification using Neural Networks

## Project Description
This project uses a Neural Network to classify handwritten digits (0-9) using the MNIST dataset.

## Technologies Used
- Python
- TensorFlow
- NumPy
- Matplotlib
- Google Colab

## Dataset
MNIST dataset containing 60,000 training images and 10,000 testing images.

## Model Architecture
- Flatten Layer
- Dense (128 neurons, ReLU)
- Dense (64 neurons, ReLU)
- Output Layer (10 neurons, Softmax)

## Accuracy
Achieved approximately 97-99% accuracy.

## Conclusion
The Neural Network successfully classifies handwritten digits with high accuracy.
